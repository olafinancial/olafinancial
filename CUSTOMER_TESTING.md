# Testing Pul Planning without deleting your account

You **do not need to delete and recreate your account** to try features again. Use the tools under **Settings**.

---

## Recommended path (clean retest)

### 1. Reset my data
1. Sign in to [pul.llc](https://pul.llc).
2. Open **Settings** (sidebar → Account → Settings).
3. Scroll to **Testing & account**.
4. Click **Reset my data** and confirm.

**What this does**
- Removes your financial records (income, expenses, assets, liabilities, goals, snapshots, etc.).
- Clears app preferences stored in the browser for testing.
- **Keeps your email and password** — you stay signed in with the same login.
- Opens the **setup wizard** so you can start fresh.

### 2. (Optional) Load demo data
After a reset (or on an empty account):

1. Stay in **Settings → Testing & account**.
2. Click **Load demo data** and confirm.

**What this does**
- Adds a **sample month** of data: salary, expenses, savings, a loan, a small equity holding, etc.
- Lets you click through **Dashboard, Budget, Balance Sheet, Debt, Reports**, and other pages without typing everything by hand.

### 3. Explore
Use the menu to open the areas you care about (Budget, Assets, Liabilities, Insurance, Calculators → Zakat, Invest Profile, etc.).

### 4. Test again later
Repeat **Reset my data** anytime you want a clean slate. Same account, no re-registration.

---

## Other useful options

| Action | Where | When to use |
|--------|--------|-------------|
| **Replay setup wizard** | Settings → Getting started | Tour the onboarding steps without wiping data |
| **Show path on Dashboard** | Settings → Getting started | Bring back the “start here” checklist |
| **Delete account** | Settings → Testing & account | Only if you truly need to **sign up again with the same email** |

**Delete account** is permanent. Prefer **Reset my data** for normal testing.  
Account deletion may require the production API server to be configured; if it is unavailable, use **Reset** instead and contact us if you still need the login removed from our auth system.

---

## Quick answers

**“I want to try the site as if I’m new.”**  
→ **Reset my data**, then either walk through the wizard or **Load demo data**.

**“I want sample numbers to click around.”**  
→ **Load demo data** (reset first if you want only sample data).

**“Do I need a special demo login?”**  
→ Not required. Your own account + reset/demo tools is enough.

**“Can I get my old numbers back after a reset?”**  
→ No — reset permanently deletes those records. Export or screenshot anything important first.
